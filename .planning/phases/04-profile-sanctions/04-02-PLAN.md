**1. Create `src/lib/utils/sanctions.ts`** -- Pure penalty calculation functions:

```typescript
// Art. 641 Estatuto Tributario Colombia
import { UVT_2026 } from "./constants";

// Exported constants for sanctions calculations
export const SANCION_MINIMA_UVT = 10;
export const SANCION_MINIMA_2026 = SANCION_MINIMA_UVT * UVT_2026; // $523,740
export const TASA_EXTEMPORANEIDAD_641 = 0.05; // 5% per month
export const TOPE_EXTEMPORANEIDAD_641 = 1.0; // 100% cap
export const TASA_MORATORIO_APROX = 0.27; // ~27% annual (approximate, conservative)

export interface PenaltyBreakdown {
  impuestoACargo: number;
  mesesExtemporaneidad: number;
  tasaMensual: number;
  subtotal: number;
  tope: number;
  minimoUVT: number;
  sancionBase: number;
  interesMoratorio: number;
  totalEstimado: number;
}

export function calculatePenaltyArt641(
  impuestoACargo: number,
  diasExtemporaneidad: number,
): PenaltyBreakdown { ... }
```

Implementation details:
- `meses = Math.max(1, Math.ceil(diasExtemporaneidad / 30))` -- Art. 641 counts any fraction of a month as a full month
- `subtotal = impuestoACargo * tasa * meses`
- `tope = impuestoACargo` (100% cap)
- If impuestoACargo > 0: `sancionBase = Math.max(Math.min(subtotal, tope), minimoUVT)`
- If impuestoACargo === 0: `sancionBase = minimoUVT` (10 UVT floor always applies)
- `interesMoratorio = impuestoACargo > 0 ? impuestoACargo * (TASA_MORATORIO_APROX / 365) * diasExtemporaneidad : 0`
- `totalEstimado = Math.round(sancionBase + interesMoratorio)`
- Round `interesMoratorio` to nearest integer

Also export a helper function for inline table use:
```typescript
export function getInlinePenaltyInfo(
  displayStatus: ObligationStatus,
  fechaVencimiento: string,
): { text: string; showLink: boolean } | null
```
- Import `daysUntilDeadline` from dates.ts and `formatCOP` from currency.ts
- If displayStatus is "PENDIENTE" or "PAGADO": return null (no warning)
- If displayStatus is "PROXIMO": return `{ text: "Sancion min. si vence: ${formatCOP(SANCION_MINIMA_2026)}", showLink: true }`
- If displayStatus is "VENCIDO": return `{ text: "Sancion est. min.: ${formatCOP(SANCION_MINIMA_2026)}", showLink: true }`
  Note: For inline display we always show the minimum (10 UVT) since we don't know the impuesto a cargo. The full calculator lets users input that value.
